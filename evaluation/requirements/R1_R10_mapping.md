# TKTOnto Evaluation Requirements Mapping

This document maps the ten evaluation requirements (R1–R10) used in the
comparative evaluation to concrete aspects of the TKTOnto framework.

| Requirement | Description | TKTOnto Support |
|------------|-------------|-----------------|
| R1 | Object identification | Stage 2: Object identification |
| R2 | Object classification (rigid/sortal) | Stage 2 with UFO meta-properties |
| R3 | Event representation | Stage 3: Event modelling |
| R4 | Causal relations | Event–state causal chains |
| R5 | Temporal ordering | Phases and situations |
| R6 | Ontological grounding | UFO-aligned OntoUML constructs |
| R7 | State evolution | Phase transitions |
| R8 | Traceability | State instantiation with time |
| R9 | Shared conceptualisation | Stage 1 elicitation |
| R10 | Reproducibility | Explicit methodology and artefacts |
